Активность клиентов

Активность клиентов

Клиенты система в Вашей установке OpenCart позволяет отслеживать и управлять учетными записями Ваших клиентов из панели администратора. Раздел доступен из Клиенты > Клиенты. Для большей информации - нажмите здесь.

Для просмотра Активности клиентов Отчеты > Клиенты > Активность клиентов:

Этот раздел покажет Вам список последних действий. У Вас также есть фильтр для всех Ваших клиентов и их активности с даты до даты. Вы можете видеть имя учетной записи каждого клиента, его IP-адрес и вид деятельности на каждую дату.

Клиент: Имя Вашего клиента.

IP-адрес: IP-адрес, используемый Вашим клиентом.

Комментарий: Вид деятельности Ваших клиентов. Например: Джон Доу зарегистрирован для учетной записи.
