Отчет о возврате

С отчетом о возвратах можно ознакомиться через Отчеты>Продажи>Возвраты. Отчет о возвратах показывает, сколько возвратов было запрошено за данный период времени. Информация о возврате отображается с ежедневными, еженедельными, ежемесячными или ежегодными приращениями в зависимости от того, какой вариант Вы выбрали в "Группировке".

Отчет о возвратах отображает следующую информацию:

Дата начала Дата первого дня недели отслеживаемого (если установлена неделя в группировке).
Дата завершения Дата последнего дня недели отслеживаемого (если установлена неделя в группировке).
Номер возврата Количество запросов на возврат, отправленных или созданных между датой начала и концом даты.
