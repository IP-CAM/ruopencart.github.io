Отчет партнерской комиссии

Отчет партнерской комиссии можно получить доступ из Отчеты>Партнерки>Комиссия в администрации. Этот отчет позволяет отслеживать филиалов, которые делают комиссию за привлечение клиентов в магазин.

Отчет партнерской комиссии отображает следующую информацию:

  • Имя партнера: Имя физического лица / компании, действующей в качестве филиала для магазина.
  • Эл. Почта: Электронный адрес для связи с партнером.
  • Статус: Показывает, включен ли партнер или отключен.
  • Комиссия: Сумма комиссионного вознаграждения, сделанного с покупки клиента.
  • Номер заказа: Количество заказов, полученных через этого партнера.
  • Итого: Общая сумма, которую этот партнер заработал через партнерскую программу.

Нажмите "Редактировать" под "Действием" направит Вас к форме партнерской информации под Продажи>Партнеры. Партнеры могут быть отфильтрованы по дате окончания и дате начала.
