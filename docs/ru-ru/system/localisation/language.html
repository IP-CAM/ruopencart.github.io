Язык локализации

Язык

Языковой раздел позволяет Вам управлять языками, доступными для магазина. OpenCart предоставляет английский как язык по умолчанию после установки. Чтобы добавить несколько языков на сайт, Вам нужно установить языковой пакет. Языковые пакеты можно приобрести в Каталог расширений. См. Добавление нескольких языков чтобы узнать, как добавить языковой пакет в OpenCart.

Если новый языковой пакет уже установлен в магазине OpenCart, новый язык можно добавить на сайт в разделе Система -> Локализация -> Языки. Нажмите кнопку   Добавить   в правом верхнем углу, чтобы добавить новый язык. Следующая информация будет необходима для заполнения.


Название языка * Установите название языка отображения в OpenCart Store.
Код * Код языка ISO. W3Schools имеет полный список кодов ISO.
Локаль * Код, используемый браузерами для автоматического определения языка сайта. Коды локали можно посмотреть здесь.
Статус Включает или отключает язык в магазине.
Порядок сортировки Определяет положение языка среди нескольких языков в заголовке.
