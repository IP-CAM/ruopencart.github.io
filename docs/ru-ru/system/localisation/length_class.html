Локализация единицы длины

Единица длины

Единица длины используются для присвоения спецификаций длины товарам в вкладка Данные. Значения, добавленные к товарам, будут отображаться на странице товара в магазине. Следующая информация необходима для создания нового единицы длины:

  • Название: Название измерения длины (т.е. сантиметр)
  • Единица: Единицы измерения (т.е. см)
  • Значение: Установите 1 в качестве значения по умолчанию. Каждое значение, добавленное к другим длинам, таким как км, должно быть относительно этого. На 1 см приходится 10 мм, поэтому значение мм будет равно 10.
