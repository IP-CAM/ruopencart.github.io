Локализация единицы веса

Единица веса

Подобно классам длины, весовые классы используются для присвоения весовых характеристик продуктам в вкладка Данные. Следующая информация необходима для создания новой весовой категории:

  • Название веса: Название измерения веса (т.е. килограмм)
  • Единица веса: Единицы измерения (т.е. кг)
  • Значение: Установите 1 для значения по умолчанию. Каждое значение, добавляемое к другим весам, таким как г, должно быть относительно этого значения. В 1 кг 1000 г, поэтому значение г будет установлено на 1000.
