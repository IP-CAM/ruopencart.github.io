Локализация геозоны

ГеоЗона

Геозоны - это настраиваемые зоны, используемые для расчета стоимости доставки и налогов. Несколько стран и зон могут быть добавлены в одну геозону, чтобы настроить способы доставки.

Геозона ниже используется для объединения западных штатов США в одну зону. Западное побережье США состоит из штатов Калифорния, Орегон и Вашингтон. Каждое отдельное состояние добавляется путем выбора страны, зоны и затем нажатия "Добавить геозону".

См. Доставка для получения дополнительной информации о том, как используются геозоны.
