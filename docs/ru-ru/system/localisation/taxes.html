Локализация налогов

Налоги

Раздел Налоги в локализации позволяет администратору создавать налоги, которые будут применяться к конкретным продуктам в магазине.

Налоговые ставки

Налоговая ставка будет использоваться для расчета налогов с заказа. Следующая информация необходима для сохранения налоговой ставки для администрации:

  • Название налога: Название для налога, которое будет отображаться на витрине магазина при добавлении налогов к общей сумме заказа.
  • Ставка налога: Числовое значение для налога
  • Тип: Определяет, является ли числовое значение в налоговой ставке фиксированной суммой или в процентах от общей суммы заказа.
  • Группа клиентов: Выбор группы клиентов добавит этот налог ко всем клиентам в этой группе. Несколько групп клиентов могут быть выбраны одновременно.
  • Геозона: При выборе геозоны налог будет применяться к регионам и странам, входящим в геозону.

Налоговый класс

Налоговые классы могут хранить несколько налоговых ставок в одной категории. Налоги могут быть настроены для расчета налогов на основе адреса платежа (покупателя) или адреса магазина. Налоговые классы требуют следующую информацию:

  • Название налогового класса
  • Описание
  • Ставка налога: выбор "Добавить Правило" добавит налоговую ставку, которая была создана в Налоговых ставках. Основывается на либо адрес оплаты, либо адрес магазина. Это определит сумму налога, добавленную к общей сумме корзины покупок в магазине. Приоритет определяет позицию налоговой ставки, если с ней указаны другие налоговые ставки.

Налоговый класс должен быть добавлен к каждому продукту индивидуально, чтобы применять налоговые ставки для покупки. Поле налогового класса находится на вкладке «Данные» в разделе «Продукты». Смотрите продукты для получения дополнительной информации.
