Локализация состояния на складе

Состояние на складе

В этом разделе Вы можете создать статус «Нет в наличии», который будет отображаться на странице продукта, когда товара нет в наличии. Название статуса акции является произвольным.

Статус запаса по умолчанию для продуктов на складе можно изменить на вкладке параметров в Настройки. Определенный статус «Отсутствует на складе» для продукта можно изменить на вкладке «Данные» в Продуктах.
