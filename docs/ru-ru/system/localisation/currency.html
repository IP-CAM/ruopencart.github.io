Локализация Валюты

Валюта

Разделы валют позволяют Вам выбрать, какие валюты доступны для использования в магазине. Валюта по умолчанию будет отображать все цены продукта в этой валюте. Предпочтение валюты может быть изменено клиентом в разделе заголовка любой страницы во фронт-офисе.

Видеоурок

В приведенном ниже магазине мы включили евро, фунт и доллар США.

Для добавления валюты в Ваш магазин необходима следующая информация:

  • Название валюты
  • Код: Код ISO для валюты. Коды валют можно посмотреть на этом конвертер валют.
  • Символ слева: Добавляет символ валюты слева от кода.
  • Символ справа: Добавляет символ валюты справа от кода.
  • Десятичные знаки: Количество знаков после запятой, отображаемое в цене магазина.
  • Значение: Для валюты по умолчанию будет установлено значение 1. Каждое значение валюты будет относиться к этому значению. Так как доллар США равен 1,0 (по умолчанию), евро сведено в таблицу к 0,81 для значения, основанного на его стоимости. Когда OpenCart осуществляет переход между валютами в магазине, он использует номер значения для расчета цены продуктов в этой новой валюте. OpenCart обновит значения валюты на вкладке «Локальные» в настройках магазина, если включена функция автоматического обновления валюты. Посетить Настройки чтобы получить больше информации.
  • Статус: Включает или отключает валюту в магазине.
