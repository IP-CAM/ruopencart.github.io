Настройки Магазин

Вкладка Магазин

На вкладке магазина в разделе «Настройки» отображаются следующие поля:

  • Заголовок: Название магазина, которое будет отображаться на панели инструментов заголовка браузера.
  • МетаТег Description: Тег, предоставляющий описание метаданных для магазина. Метатеги видятся поисковыми системами.
  • Шаблон: Выбирает шаблон / тему магазина. OpenCart имеет только один, "По умолчанию". Themeforest is a great resource for discovering 3rd party templates for OpenCart.
  • Макет по умолчанию: Тип макета по умолчанию для магазина.
