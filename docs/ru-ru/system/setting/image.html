Settings Image

Image Tab

The Image tab under Settings customizes the default images for the store:

  • Store Logo: Upload a store logo from the store or the admin's computer using Image manager.
  • Icon: Upload a favicon for the store using Image manager.
  • Image Sizes: Lets you customize the pixel width by length for specific images in the store front. Image sizes are now managed at the theme level. For the default Opencart theme, these settings can be found in the Opencart admin under Extensions->Themes(Extension Type Dropdown) then Edit the Default store theme.
