Настройки Изображение

Вкладка Изображение

На вкладке «Изображение» в разделе «Настройки» настраиваются изображения магазина по умолчанию:

  • Логотип магазина: Загрузите логотип магазина из магазина или с компьютера администратора, используя Менеджер изображений.
  • Иконка: Загрузить фавикон для магазина, использующего Менеджер изображений.
  • Размеры изображения: Позволяет настроить ширину пикселя по длине для конкретных изображений в витрине магазина. Размеры изображений теперь управляются на уровне темы. Для темы Opencart по умолчанию эти параметры можно найти в администраторе Opencart в разделе «Расширения -> Темы» (раскрывающийся список «Тип расширения»), а затем «Изменить тему хранилища по умолчанию».
