Настройки Мошенничество

Вкладка Мошенничество

На вкладке «Мошенничество» в разделе «Настройки» можно зарегистрироваться, чтобы использовать систему обнаружения мошенничества MaxMind для OpenCart. Посетить MaxMind для получения дополнительной информации.
