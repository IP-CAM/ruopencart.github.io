Настройки Общие General

Вкладка Общие

На вкладке «Общие» в разделе «Настройки» отображаются следующие поля:

  • URL магазина: URL магазина. Не забудьте включить 'http:' в начале и '/' в конце.
  • Название магазина: Название магазина. Эта информация будет отображаться на странице «Контакты» в разделе «Обслуживание клиентов» нижнего колонтитула.
  • Владелец магазина: Имя, используемое в "От:" раздел любого электронного письма, отправленного из магазина.
  • Адрес: Адрес магазина. Эта информация будет отображаться на странице «Контакты» в разделе «Обслуживание клиентов» нижнего колонтитула.
  • Эл. почта: Электронная почта владельца магазина или общая электронная почта для магазина, которая будет использоваться для уведомлений по электронной почте, отправляемых OpenCart.
  • Телефон: Телефонный номер для магазина. Эта информация будет отображаться на странице «Контакты» в разделе «Обслуживание клиентов» нижнего колонтитула.
  • Факс (необязательно): Номер факса для магазина. Эта информация будет отображаться на странице «Контакты» в разделе «Обслуживание клиентов» нижнего колонтитула..
