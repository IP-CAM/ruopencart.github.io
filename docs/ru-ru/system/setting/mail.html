НАстройки Почта

Вкладка Почта

На вкладке «Почта» в разделе «Настройки» отображаются следующие поля:

  • Почтовый протокол: Позволяет выбирать между отправкой почты на электронную почту хоста или через SMTP. Вы должны связаться с хостинг-провайдером Вашего магазина, чтобы узнать, настраиваете ли вы почту PHP Если нет, Вам нужно использовать SMTP, например, Gmail.
  • Параметры почты: Запрашивает адрес электронной почты для магазина, если "Почта" был выбран в разделе Почтовый протокол.
  • SMTP Хост: Адрес хоста SMTP.
  • SMTP Имя пользователя: Полный адрес электронной почты для узла SMTP
  • SMTP Пароль
  • SMTP Порт: Номер порта, предоставленный узлом SMTP.
  • SMTP Тайм-аут: Номер таймаута, предоставленный узлом SMTP.
  • Оповещение о новом заказе: При выборе 'Да' будет отправлено уведомление по электронной почте для каждого заказа, представленного в магазине.
  • Оповещение о новой учетной записи: При выборе 'Да' отправит уведомление по электронной почте для каждой учетной записи, созданной в магазине.
  • Дополнительные оповещения по электронной почте: Перечислите любые дополнительные электронные письма, которые также должны получать уведомления по электронной почте выше.
