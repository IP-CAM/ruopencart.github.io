Бонусные баллы продукта

Вкладка Бонусные баллы

Бонусные баллы - это особенность OpenCart, которая присваивает клиентам "очки лояльности" для покупки товаров в магазине. Клиенты могут использовать полученные баллы в качестве валюты для покупки товаров в магазине. Вы можете назначить этому продукту определенное количество бонусных баллов, необходимых клиенту для его покупки. Если Вы не хотите, чтобы этот продукт покупался с бонусными баллами, Вы можете просто оставить это поле пустым.
