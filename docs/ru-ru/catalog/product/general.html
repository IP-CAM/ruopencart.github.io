Общие сведения о продукте

Вкладка Общие

Формат вкладки «Общие» идентичен вкладке «Общие» при создании категории продукта с добавлением функции теги товара:

  • Название продукта
  • МетаТег Description: Резюме, чтобы описать страницу продукта для поисковых систем.
  • МетаТег Keywords: Аналогичен МетаТегу Description, но запрашивает одно ключевое слово для описания продукта.
  • Описание: Текст, который будет описывать этот продукт на странице продукта, чтобы быть просмотренными покупателями в магазине перед магазином.
  • Теги продуктов: Тег для описания продукта для клиентов. Если клиент введет этот тег в поле поиска, найденное в заголовке, продукт будет отображаться в результатах поиска..
