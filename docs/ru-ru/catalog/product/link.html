Связи продукта

Вкладка Связи

Вкладка Связи указывает, на каких страницах OpenCart будет отображаться продукт:

  • Производитель: Выбор производителя товара разместит товар на странице производителя. Выбор производителей можно посмотреть на главной странице, если они добавлены в баннер производителя.
  • Категории: Проверьте категории, в которые продукт будет помещен. Товар будет указан на странице категории в магазине.
  • Фильтры: Выберите фильтры, которые будут применяться к этому продукту, если ваш модуль фильтра включен. Смотрите Фильтры для получения дополнительной информации.
  • Магазины: Проверьте, какие магазины будут продавать этот продукт. Нажатие "По умолчанию" применяется, если есть только один магазин для управления.
  • Загрузки: Если продукт является загружаемым продуктом, выберите файл, который будет включен в покупку.
  • Сопутствующие продукты: В этом месте введите любые связанные продукты для этого продукта. Эти продукты будут отображаться на странице этого продукта для просмотра клиентом. При вводе в поле появится раскрывающееся окно со связанными продуктами.
