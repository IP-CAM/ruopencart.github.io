Скидка продукта

Вкладка Скидка

На вкладке «Скидка» требуется следующая информация, чтобы включить скидку на продукт:

  • Группа клиентов: Группа клиентов (сделано в Продажи > Клиенты > Группы) которые соответствуют критериям для использования этой скидки.
  • Количество: Лимит на количество товаров, на которые может распространяться данная скидка.
  • Приоритет: Добавление номера приоритета, такого как 1, 2 или 3, определит, когда эта скидка будет использоваться при применении других скидок к заказу клиента. 1 применяет эту скидку первой, 2 применяет ее второй, и так далее.
  • Цена: Цена со скидкой.
  • Дата начала: На первую дату будет применена скидка.
  • Дата завершения: Последняя дата, доступная для использования скидки; дата окончания срока действия скидки.

Нажмите "Добавить Скидку" после заполнения информации. Нажмите "Удалить" для удаления скидки с товара.
