Варианты продукта

Вкладка Опции

Опции - это дополнительный выбор, который клиент может сделать на странице «Продукт» перед тем, как добавить продукт в корзину. Параметры сначала создаются в Каталог > Опции, затем присваивается продукту, который создается в Каталог > Продукты.

Одним из примеров варианта является "размер". Начните вводить "размер" в поле поиска. Размер должен появиться в выпадающем списке. Выберите его и нажмите зеленую кнопку добавления. Справа нажмите "Добавить значение опции".

Значения параметров ранее были созданы в Опции раздел под каталогом. В этом разделе можно добавить любую опцию, такую как "Большой". Чтобы вычесть из количества каждый раз, когда продукт приобретается с помощью этой опции, вы можете выбрать "Да" под "Вычитать со склада". Повышение цены может быть добавлено к значению опции, баллам и весу. Нажмите "Добавить значение опции" назначить эту информацию конкретному значению опции. Вы можете нажать "Удалить" стереть значение параметра. См. Опции для более глубокого объяснения функции опций.
