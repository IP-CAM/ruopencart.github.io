Данные о продуктах

Вкладка Данные

Вкладка данных включает в себя все критерии продукта, которые будут отображаться на странице продукта в витрине магазина для просмотра покупателями:

Заполните вышеприведенные поля следующей информацией:

  • Модель: создайте номер модели для продукта, используя комбинацию букв и цифр. (обязательно)
  • SKU (складская единица): случайный код для продукта.
  • UPC (универсальный код продукта): уникальный штрих-код продукта.
  • Расположение: где находится продукт.
  • Цена: установить цену на товар.
  • Налоговый класс: выберите категорию налогового класса для этого продукта, который был сделан в Настройки > Локализация > Налоговые классы. Это будет применять налог в кассе для продукта. Выберите "Ничего" если к этому продукту не будут применяться налоги.
  • Количество: Общее количество, доступное в магазине для этого конкретного продукта.
  • Минимальное количество: минимальное количество товара, которое клиент должен достичь, чтобы добавить этот товар в свою корзину.
  • Вычитать со склада: "Да" вычтет со склада из количества продукта (например, если имеется 100 компьютеров, а клиент покупает 2 компьютера, Вычитать со склада изменит количество до 98).
  • Статус Нет на складе: Выбор "Нет на складе", "В наличии", "2-3 дня" или "Предварительный заказ" как сообщение, отображаемое на странице продукта, когда количество продукта достигает 0.
  • Требуется доставка: Если продукт требует доставки, выберите "Да". Если нет, выберите "Нет".
  • ЧПУ: создать ключевое слово для поисковой оптимизации этого продукта. Это будет видно в URL страницы продукта. Поэтому созданное ключевое слово должно быть уникальным для каждого продукта.
  • Изображение: нажмите "Обзор папок" приведет администратора к Менеджер Изображений для загрузки нового изображения.
  • Дата поступления: дата, когда этот продукт станет доступным для покупки.
  • Размеры: введите длину по ширине по высоте изделия, если есть размеры для включения.
  • Единица длины: определяет метрические единицы для указанных выше размеров.
  • Вес: Введите число для веса.
  • Единица веса: Укажите единицы веса для числа, введенного в "Вес".
  • Статус: Включение делает продукт общедоступным в магазине. Отключение позволяет редактировать продукт в администрации, но скрывает его от магазина .
  • Порядок сортировки: Когда продукт отсортирован в списке, номер присваивает ему приоритет. Продукт с порядком сортировки или 2 будет размещен выше, чем продукт с порядком сортировки 3, но ниже, чем продукт с порядком сортировки 1.
