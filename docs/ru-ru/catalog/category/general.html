Категория Общие

Вкладка Общие

На вкладке Общие запрашивается основная информация о категории продукта:

Название категории * -
Описание Текст, который будет описывать эту категорию на странице категории, которая будет просматриваться покупателями в магазине.
МетаТег Title * Название категории для отображения в разделе тегов браузера.
МетаТег Description Сводка категории для описания страницы категории для поисковых систем.
МетаТег Keywords Аналогичен метатегу описания, но запрашивает одно ключевое слово для описания продукта.
* Обязательное поле
