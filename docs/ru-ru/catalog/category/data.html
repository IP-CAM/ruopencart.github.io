Категория Данные

Вкладка Данные

На вкладке Данные запрашиваются следующие данные:

  • Родительская категория: Категории расположены в иерархической структуре; с родительской категорией всегда на вершине. Вы можете создать категорию в родительской категории, превратив ее в подкатегорию. В списке категорий категория будет отображаться как "Родительская категория > Подкатегория". Если ничего не выбрано, созданная категория будет родительской.
  • Фильтры: Сопоставьте все предварительно созданные фильтры для категории. См. Фильтр для получения дополнительной информации о том, как работают фильтры.
  • Магазины: Если Вы управляете несколькими магазинами с помощью OpenCart, Вы можете проверить, в каких магазинах Вы хотите добавить эту категорию. Если доступен только один магазин, "По умолчанию" может быть проверено. См. Создание мульти-магазина для получения дополнительной информации о мульти-магазинах.
  • ЧПУ: Ключевое слово, которое будет включено в URL-адрес категории для поисковой оптимизации.
  • Изображение: Выберите основное изображение для страницы продукта и эскиз для списков продуктов. См. Менеджер изображений для получения дополнительной информации о том, как загрузить изображение с помощью инструмента менеджера изображений.
  • Меню: Проверка этого раздела будет отображать эту категорию в верхнем меню каждой страницы. Если выбор не сделан, он не будет отображаться.
    В верхнем меню отображаются только родительские категории. Если эта категория является подкатегорией, она не будет отображаться в верхнем меню.
  • Столбцы: Этот номер определяет, сколько подкатегорий отображается при выборе родительской категории.
    Владельцы магазинов со многими подкатегориями продуктов, возможно, должны обратить особое внимание на то, сколько столбцов разрешено. Если все доступные подкатегории разрешены для категории продукта, вся страница магазина в витрине магазина будет заполнена подкатегориями при наведении.
  • Порядок Сортировки: Этот порядок сортировки определяет положение категории в меню. Порядок сортировки 1 будет отображаться перед порядком сортировки 2, 3 и т.д.
  • Статус: Выбор "Включено" делает категорию общедоступной на витрине магазина. Выбор "Отключено" скрывает категорию из магазина, но все еще будет доступна для редактирования в администрации.
