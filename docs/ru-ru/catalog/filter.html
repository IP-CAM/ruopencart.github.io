Фильтр

Эта функция может быть найдена на Категории и Продукты страницы при создании или редактировании на вкладке Данные.

Система по умолчанию не содержит данных о фильтре, поэтому всплывающих данных на поле нет.

Откройте левое меню в Каталог и нажмите Фильтры на четвертом ряду. На рисунке есть пустой список, нажмите создать кнопка для перехода на следующую страницу.

Зайдите на информационную страницу, там есть часть полей, которые запрашиваются:

Группа фильтров

Имя группы фильтров * Название группы фильтров
Порядок сортировки Это повлияет на сортировку групп фильтров на витрине магазина. Можно оставить это поле пустым или ввести 0.
* Поле запрошено, не может быть пустым и отправлено

Значения фильтра

Имя фильтра * Название значения фильтра
Порядок сортировки Это повлияет на сортировку значений в группе на витрине магазина. Можно оставить это поле пустым или ввести 0.
* Поле запрошено, не может быть пустым и отправлено

Редактировать категорию

Например, создается группа фильтра Цвета, и редактируйте Desktop из категорий. Добавьте Красный и Черный в эту категорию и сохраните изменения.

Редактировать продукт

Теперь отредактируйте продукт в разделе Desktop, добавьте в него фильтр Красный и сохраните изменения.

Если модуль фильтров включен, он будет отображаться в Desktop на странице передней категории магазина, когда это будет сделано. Для получения более подробной информации о настройке макета, пожалуйста, нажмите здесь.

Витрина

Если модуль установлен сверху, результат будет таким же, как на рисунке ниже при нажатии на Desktop.
