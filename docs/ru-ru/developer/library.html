Библиотека

Классы Библиотеки загружаются с помощью загрузчика.

Кеш

Кеширование обработанных данных.

  • __constuct(string $adaptor, [int $expire = 3600])
  • get(string $key) : (mixed) - получает данные кеша по ключу
  • set(string $key, mixed $value) - кэширует данные с помощью ключа
  • delete(string $key) - удаляет кеш по ключу
