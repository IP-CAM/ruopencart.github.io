Система Расширений

В версии 3.0 мы добавили некоторые дополнительные ограничения пути, чтобы устанавливаемые расширения не вызывали проблем при удалении.

Система Расширений

Типы Расширений

  • analytics (аналитика)
  • captcha (капча)
  • dashboard (инфо панель)
  • extension (расширения)
  • feed (каналы)
  • fraud (мошенничество)
  • module (модули)
  • payment (оплата)
  • report (отчеты)
  • shipping (доставка)
  • theme (темы)
  • total (итого)

Разрешенные файлы расширений каталогов могут быть записаны в:

Admin

  • admin/controller/extension/
  • admin/language/
  • admin/model/extension/
  • admin/view/image/
  • admin/view/javascript/
  • admin/view/stylesheet/
  • admin/view/template/extension/

Catalog

  • catalog/controller/extension/
  • catalog/language/
  • catalog/model/extension/
  • catalog/view/javascript/
  • catalog/view/theme/

System

  • system/config/
  • system/library/

Image

  • image/catalog/
