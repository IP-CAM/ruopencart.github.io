Помощники

Помощники - это PHP-функции без сохранения состояния. В OpenCart они используются для исправления некоторых недостатков некоторых встроенных функций PHP. Например, функция strtolower не может обрабатывать много иностранных символов, а функции UTF-8, которые устанавливаются только в более поздней версии PHP.
