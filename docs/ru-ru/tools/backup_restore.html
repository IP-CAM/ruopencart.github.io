Резервирование/Возвращение

Резервное копирование и восстановление

Одним из самых разрушительных событий, которые могут произойти с интернет-магазином, является потеря базы данных. База данных магазина содержит важную информацию о клиентах и продуктах. Восстановление магазина после потери базы данных затруднено, и, безусловно, отнимает слишком много времени для повторного ввода этой необходимой информации. Вы должны быть бдительными при загрузке обновленных резервных копий базы данных магазина на их компьютер, на случай, если что-то случится.

OpenCart позволяет загрузить резервную копию текущей базы данных магазина или восстановить базу данных для администрирования в Система>Резервное копирование/Восстановление. В разделе «Резервное копирование», представленном ниже, содержатся все таблицы данных в базе данных MySQL, доступные для скачивания. Вы можете выбрать, какие файлы Вы хотите загрузить, проверив их. Нажав на кнопку Экспорт загрузит выбранные таблицы на Ваш компьютер.

Если Вам нужно восстановить файл резервной копии, который Вы ранее сохранили на свой компьютер, все, что Вам нужно сделать, это нажать Импорт в Резервное копирование и восстановление выберите файл sql со своего компьютера, и база данных будет восстановлена.
