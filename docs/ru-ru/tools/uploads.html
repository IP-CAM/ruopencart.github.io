Загрузки

Загрузки

Чтобы открыть раздел «Загрузка» в панели администратора OpenCart, нажмите Система > Техническое обслуживание > Загрузки:

В этом поле отображаются все загруженные файлы, которые есть в Вашем магазине OpenCart. Это могут быть изображения, файлы PDF и т.д.

У Вас есть кнопка фильтра, которая сортирует загруженные файлы в соответствии с именем загрузки и датой добавления.

Кнопка «Действие» с правой стороны позволяет Вам загрузить загруженный файл из Вашего магазина OpenCart. Соответственно, кнопка Удалить позволяет удалить выбранный файл.
