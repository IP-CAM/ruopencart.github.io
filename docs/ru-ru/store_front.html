Просмотр магазина

Просмотр витрины магазина

Это руководство предназначено для использования в качестве введения в стандартную витрину магазина OpenCart. Витрина магазина показывает, как клиент просматривает и взаимодействует с магазином.

Тема по умолчанию

OpenCart поставляется с темой по умолчанию после новой установки:

Представленные выше продукты включены в качестве примера данных при установке OpenCart. Эти продукты могут быть легко удалены и заменены продуктами магазина позже.

В этом руководстве будут рассмотрены основы просмотра магазина с темой по умолчанию. Вы можете найти постоянно расширяющийся список доступных тем в OpenCart Каталог Расширений.

Навигация по магазину

Тема по умолчанию OpenCart делает навигацию по продуктам магазина легко доступной для клиентов.

Анатомия домашней страницы

Домашняя страница является, пожалуй, самой важной страницей в магазине, с точки зрения презентации. В большинстве случаев это будет первая страница, с которой взаимодействует покупатель (особенно, если они направлены на сайт магазина из поисковой системы). Домашняя страница магазина должна быть удобной для пользователя, и в то же время подчеркивать продукцию магазина.

Первым шагом в ознакомлении с витриной магазина является понимание анатомии домашней страницы OpenCart по умолчанию.

Заголовок

Заголовок будет отображаться в верхней части страницы, на каждой странице магазина; не только домашняя страница.

Заголовок имеет следующие параметры навигации:

  • Логотип магазина: при нажатии на этот логотип покупатель возвращается на главную страницу магазина.
  • Блок валюты: покупатель может выбрать валюту, в которой будут находиться товары в магазине, щелкнув любую иконку валюты.
  • Корзина покупок: отображает количество приобретенных товаров и общую стоимость заказа. Нажатие на кнопку будет содержать все товары, добавленные в корзину и возможность "Просмотра корзины" или "Оформления заказа".
  • Поле поиска: клиенты могут ввести в поле поиска, чтобы найти товар в товарных категориях магазина.
  • Ссылки: связывает клиента с домашней страницей, списком пожеланий, учетной записью, корзиной покупок и оформлением заказа.
  • Телефон: Телефон компании.
  • Моя учетная запись: Клиент может зарегистрироваться или войти здесь.

Верхнее меню

В верхнем меню категории меню отображаются только верхние родительские категории продуктов. Смотрите Категории для получения дополнительной информации о том, как создавать и назначать категории продуктов.

Как и заголовок, верхнее меню будет отображаться на каждой странице. Когда указатель мыши перемещается над категорией, в раскрывающемся меню отображаются подкатегории для этой родительской категории.

При щелчке по родительской категории покупатель будет перенаправлен на страницу категории, где отображаются все продукты в этой категории.

Слайд-шоу

Слайд-шоу отображает несколько баннеров продукта по Вашему выбору, чередуя изображения в слайд-шоу. Через некоторое время один баннер перейдет к следующему баннеру. Баннеры в этом слайд-шоу полезны для выделения определенных продуктов, которые будут легко доступны для покупателя. При нажатии на баннер, клиент будет перенаправлен на товар на странице баннера.

В отличие от верхнего меню и заголовка, слайд-шоу в OpenCart по умолчанию можно просматривать только на главной странице в этой позиции.

Рекомендуемые товары

OpenCart дает Вам возможность размещать определенные продукты по своему выбору на главной странице.

Избранный раздел включает изображение товара, название, цену; и возможность добавить товар прямо в корзину.

Карусель находится только на домашней странице по умолчанию.

Футер

Нижний колонтитул расположен внизу каждой страницы, а не только домашней страницы. Этот блок разных ссылок полезен для сортировки релевантных страниц для клиентов, которые не могут логически сортироваться где-либо еще.

Организационную схему футера можно разделить на следующие разделы:

  • Информация: "О нас", "Информация о доставке", "Политика конфиденциальности", "Условия & Положения"
  • Обслуживание клиентов: "Связаться с нами", "Возвраты", "Карта сайта"
  • Дополнительно: "Бренды", "Подарочные сертификаты", "Партнерская программа ", "Специальные предложения"
  • Личный кабинет: "Личный кабинет", "История заказов", "Закладки", "Рассылка"

Страницы продукта

Страница продукта OpenCart по умолчанию будет соответствовать структурному формату, показанному ниже.

Страница продукта может быть разделена на следующие разделы:

  • Изображение продукта: Изображение продукта может отображаться под заголовком слева, а также альтернативные виды продукта под ним в меньшей рамке. Нажатие на основное изображение раскроет изображение в окне, чтобы клиент мог увидеть его более подробно.
  • Информация о продукте: Код продукта, наличие и цена отображаются прямо справа от изображений продукта.
  • Корзина покупок: Клиент может выбрать количество и добавить товар в корзину, список желаний или сравнить.
  • Рейтинг/Поделиться: Под корзиной можно оценить продукт и/или поделиться продуктом на разных сайтах в социальных сетях.
  • Вкладка "Описание": Область под основной информацией о продукте, чтобы предоставить подробное описание продукта.
  • Вкладка «Отзывы»: Область для клиента, чтобы написать отзыв о продукте.

Список продуктов категории

Списки продуктов категории позволяют клиентам просматривать продукты, похожие на другие продукты в той же категории. Это особенно полезно для клиентов, которые хотят сравнивать продукты. Эта функция будет описана в разделе Категории. Доступ к странице категории можно получить несколькими способами. Доступ к нему можно получить из верхнего меню, когда клиент нажимает на одну из родительских категорий. Кроме того, на страницах продукта клиент может получить доступ к странице со списком продуктов категории, нажав на категорию в левой части блока категорий.

Как видно выше, блок категорий отображается слева, как на странице продукта. Под заголовком «Категория» вверху есть место для добавления описания в категорию. "Уточненный поиск" ссылки на подкатегории этой категории для просмотра пользователем. Продукты могут отображаться в соответствии с предпочтениями клиента: в виде списка или сетки. Изображение выше отсортировано в формате списка. Продукты могут быть отсортированы по названию, цене, рейтингу или модели в поле "Сортировать по". Количество товаров, отображаемых в списке товаров, может быть изменено в "Показать по" от 15 до 100.

Есть раздел, который дает место для каждого из продуктов в категории, предоставляя изображение продукта, описание, цену и опцию Добавить в корзину. Есть возможность добавить товар в список желаний. Другим вариантом для продукта является "Добавить к сравнению".

Сравнение продуктов

"Добавить к сравнению" в разделе продукта позволяет клиенту сравнивать различные характеристики, функции и цены ряда продуктов, которые могут быть ему интересны.

Клиенту предоставляется возможность добавить один из сравниваемых продуктов в корзину, если он этого хочет. Нажатие кнопки "Продолжить" вернет пользователя на домашнюю страницу.

Страница корзины покупок

Как только покупатель добавляет товар в корзину, он может получить доступ к корзине покупок в шапке под "Корзиной покупок".

Корзина покупок дает обзор выбранного продукта, включая категории "Изображение", "Наименование продуктов", "Модель", "Количество", "Цена за единицу", и "Общее". У клиента есть возможность добавить код купона или подарочный сертификат, или оценить стоимость доставки и налоги, прежде чем отправиться к оформлению заказа. Кнопка "Продолжить покупки" ссылается на домашнюю страницу.

Создание аккаунта

Прежде чем клиент сможет продолжить извлечение товара из корзины покупок, ему необходимо выбрать либо оформить заказ гостя, либо войти в свою учетную запись. Гостевой заказ не требует входа в систему. Постоянные клиенты могут захотеть создать аккаунт в магазине.

Клиент может создать учетную запись несколькими способами:

  1. Оформление заказа

    Шаг 1 процесса проверки позволяет пользователю создать учетную запись, прежде чем продолжить оплату. Выбор "Зарегистрировать аккаунт" изменит Шаг 2 оформления заказа с выставления счета на учетную запись и платежные данные. Учетная запись и биллинг запрашивает те же личные данные, что и биллинг, за исключением того, что он просит пользователя создать пароль для своей учетной записи. После завершения шага 2 клиент может продолжить процесс оформления заказа.

  2. Шапка- "Личный кабинет"

    Если кликнуть "Личный кабинет" в шапке, клиенту будет предложено войти в систему или создать учетную запись. На этой странице клиент может войти в систему, если у него уже есть учетная запись, или создать новую учетную запись. В разделе "Новый Клиент" может кликнуть "Продолжить" под Регистрацией Аккаунта для редиректа на страницу"Зарегистрировать Аккаунт".

Оформление заказа

После добавления товара в корзину покупатель может перейти к оформлению заказа, чтобы совершить покупку товара. Страница «Оформить заказ» доступна в разделе заголовка каждой страницы (находится под окном поиска). Проверка клиента с помощью OpenCart - это простой процесс, который можно выполнить за 6 шагов.

  1. Шаг 1: Варианты оформления заказа

    Клиент может войти в систему или зарегистрировать свою учетную запись (как описано выше) или выбрать гостевую проверку.

  2. Шаг 2. Платежные реквизиты

    Личные данные, включая "Имя", "Фамилию", "Эл. почта", и "Телефон" заполняются в форме. Также требуются детали адреса клиента.

  3. Шаг 3: Детали доставки

    В разделе «Платежные реквизиты» пользователь может установить флажок, чтобы указать, что данные о доставке и платежные данные совпадают. Это заставит его пропустить этот шаг до способа доставки. Если условия доставки отличаются от платежных данных, клиент может ввести эту информацию в форму в этом разделе.

  4. Шаг 4: Способ доставки

    Способ доставки выбран здесь. Поле для комментариев добавлено для клиента, чтобы добавить комментарии о своем заказе.

  5. Шаг 5: Способ оплаты

    Здесь клиент выбирает способ оплаты и может добавлять комментарии в поле для комментариев.

  6. Шаг 6: Подтверждение заказа

    На этом последнем шаге клиент увидит обзор своей покупки; включая описание продукта, количество и цену (с учетом налогов и стоимости доставки).

Сохранение настроек с использованием панели администратора

Все функции, перечисленные выше, могут быть в определенной степени настроены в панели администратора. Администратор может изменять положение определенных продуктов, отключать категории, редактировать цены и описания, загружать баннеры и т.д. В панели администратора OpenCart можно проделать некоторую работу, чтобы установить бренд магазина.
