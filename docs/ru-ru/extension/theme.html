Магазин Темы

Тема может быть установлена

Тема была сгруппирована в список расширений, начиная с OpenCart 2 (версия 2.3). Каждая тема может быть отредактирована измерение продукта, ограничение отображения продукта и длина описания. Пользователь может перейти к Официальная торговая площадка OpenCart купить больше тем для стилизации магазина.

Установка & Включить тему

Метод установки такой же, как расширение, а загрузка xxxx_theme.ocmod.zip, перейти Установщик и загрузить ZIP-файл.

Место расположения Модуль / Элемент
Расширения Установщик Загрузить xxxx_theme.ocmod.zip на странице установщика

После успешной загрузки файла перейдите в раздел Расширение и выберите Темы в раскрывающемся списке.

Место расположения Модуль / Элемент
Расширения Расширения Темы Выбрать Темы в раскрывающемся списке

Новая тема будет отображаться в списке. Нажмите кнопку установки и можно начать редактирование темы, выбрать каталог и отредактировать настройки деталей.
