Борьба с мошенничеством

Борьба с мошенничеством

Модули Борьба с мошенничеством доступны в разделе Расширения > Борьба с мошенничеством. Основные модули по борьбе с мошенничеством, представленные ниже, включены в пакет OpenCart.

Модули Anti-Fraud необходимо установить, прежде чем их можно будет включить в магазине. Процесс установки аналогичен установке расширения в разделе Расширения > Модули в OpenCart. Например, это как установить модуль FraudLabs Pro. Сначала установите модуль, нажав кнопку «Установить» под столбцом «Действие». Зеленая полоса сообщит Вам, что модуль успешно изменен.

После установки модуля Вы можете редактировать настройки модуля. Начнем с настройки первого модуля FraudLabs Pro.

FraudLabs Pro

Модуль FraudLabs Pro использует Fraudlabs Pro сервис по обнаружению возможных мошеннических заказов. Чтобы использовать сервис Fraudlabs Pro, сначала нужно получить ключ API и указать его в настройках модуля OpenCart.

Для получения ключа API необходимо зарегистрироваться в сервисе FraudLabs Pro. Есть несколько тарифных планов, в том числе бесплатный. Различные планы предлагают разное количество ежемесячных запросов (заказов) и разное количество правил проверки. Правила проверки - это настраиваемые правила, которые Вы можете настроить в дополнение к другим проверкам, которые FraudLabs Pro выполняет при проверке заказа.

После настройки и включения модуля в OpenCart каждый заказ, сделанный в Вашем магазине, будет проходить через систему FraudLabs и проверяться на предмет возможного мошенничества. После проверки заказа в сервисе FraudLabs Pro ему присваивается определенный статус и оценка. Существует три возможных статуса проверки на мошенничество: Утверждено, Отклонено или Ожидает проверки. Счет - это значение от 0 до 100, представляющее вероятность того, что заказ может быть мошенническим. Чем больше число, тем более рискован заказ.

Модуль OpenCart FraudLabs Pro настраивается на основе баллов и статуса Fraudlabs Pro для каждого заказа.

Оценка риска

После проверки каждому заказу присваивается определенный балл. Вы можете проверить, какой счет у каждого заказа, в пользовательской панели FraudLabs Pro.

В модуле OpenCart FraudLabs Pro Вам необходимо настроить показатель риска, который будет сравниваться с показателем заказа от службы FraudLabs Pro. Также в модуле должен быть настроен статус заказа. Этот Статус заказа будет применен к заказу в Вашем магазине в случае, если Оценка (полученная из сервиса FraudLabs Pro) больше, чем Оценка риска (настроенный в модуле OpenCart). Вы можете выбрать из всех доступных статусов заказов OpenCart.

Статус

Исходя из статуса результатов службы FraudLabs, заказ будет иметь один из следующих статусов: Одобрено, Отклонено или Ожидает рассмотрения.

В модуле OpenCart FraudLabs Pro Вы должны указать, какой статус заказа будет применяться к каждому заказу для каждого из статусов заказов на обслуживание FraudLabs Pro. Например, если FraudLabs Pro присваивает заказу статус Отклонено, статус заказа OpenCart можно изменить на "Отказано". Для трех сервисов FraudLabs Pro необходимо применять разные статусы заказов.

Чтобы протестировать конфигурацию модуля, Вы можете использовать поле Simulate IP. Если Вы добавите IP-адрес в это поле, то все заказы, сделанные в Вашем магазине, будут зарегистрированы с этим IP для службы FraudLabs Pro.

Дополнительные функции можно найти на панели управления с панели FraudLabs Pro:https://www.fraudlabspro.com/merchant/dashboard

Оценка риска

Вторым модулем защиты от мошенничества в OpenCart является IP-адрес защиты от мошенничества. Этот модуль обрабатывает возможные мошеннические заказы на основе IP-адресов клиентов. Модуль позволяет вручную создать список мошеннических IP-адресов. Список IP-адресов мошенников настраивается с: IP-адреса защиты от мошенничества > IP-адреса > Список IP-адресов мошенников.

В OpenCart есть журнал IP-адресов для каждого клиента. Если в списке IP-адресов клиентов есть IP-адрес мошенника, то заказ не будет автоматически менять свой статус на завершенный. Также модуль изменит статус заказа на ранее настроенный в Анти-Мошенничество IP > Общие > Статус заказа.

MaxMind Anti-Fraud

Вторым сервисным модулем обработки мошенничества в OpenCart является MaxMind Anti-Fraud. Для использования модуля Вам необходимо получить лицензионный ключ от MaxMind minFraud сервис и разместите его в настройках модуля OpenCart.

Клиенты сервиса minFraud передают в систему minFraud информацию о сервисе, предоставляемую посетителями сайта во время онлайн-транзакций и регистрации аккаунта. Служба minFraud анализирует эту информацию и возвращает оценку риска. Оценка риска варьируется от 0,01 до 100,00 и отражает вероятность того, что данная транзакция является мошеннической. Как только заказ размещен в Вашем магазине OpenCart, транзакция заказа сканируется в системе minFraud. После проверки каждому заказу присваивается определенная оценка риска. Вы можете проверить, какой счет у каждого заказа, на пользовательской панели minFraud (история запросов minFraud).

В модуле OpenCart MaxMind Anti-Fraud Вы должны настроить показатель риска, который будет сравниваться с показателем риска заказа от сервиса minFraud. Кроме того, статус заказа должен быть установлен в модуле. Этот статус заказа будет применен к заказу в Вашем магазине в случае, если Оценка риска (полученная из сервиса minFraud) больше, чем Оценка риска ( настраивается в модуле OpenCart). Кроме того, заказ не сможет автоматически достичь полного статуса.

Более подробную информацию о сервисе minFraud вы можете найти здесь: https://www.maxmind.com/MaxMind_WhitePaper.pdf
