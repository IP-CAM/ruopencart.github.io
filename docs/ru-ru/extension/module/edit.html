Редактирование модуля

Модули могут сильно различаться по назначению и функциям, поэтому, естественно, настройка модуля для конкретного магазина может варьироваться между модулями. В общем, в разделе «Модули» можно использовать 4 основных функции модулей:

Макет Позиция
Статус Порядок сортировки

Макет

Макет влияет на то, на какие страницы будет включен модуль. Доступные страницы перечислены на скриншоте ниже. Если модуль должен быть включен на нескольких страницах, Вы должны индивидуально "Добавить модуль" для каждой страницы. Макет может выбрать только одну страницу за раз.

На приведенном ниже снимке экрана показан модуль Featured, который отображает список указанных продуктов в левой части страницы со случайной категорией. Это было достигнуто путем выбора "Категория" под Макетом. Поле «Избранные товары» будет доступно на каждой странице категории. Выбор типа страницы макета добавит его на каждую страницу этого типа. Основные модули могут быть назначены страницам макета, но не отдельным страницам.

Позиция

Выбор позиции определяет, где модули будут отображаться на странице. На типах страниц, выбранных в макетах, блок модуля может находиться сверху, снизу, слева или справа от страницы.

Используя тот же модуль из раздела «Макет», мы изменим поле «Избранные» на странице категории с левого столбца вверх, выбрав "Content Top" под позицией (ниже).

Поле Featured с указанными продуктами отображается над категорией продуктов на странице категории, перемещенной из исходного положения слева.

Статус

Отключение или включение статуса модуля повлияет на видимость модуля на веб-сайте магазина. Отключение модуля отличается от удаления модуля. Отключение модуля сохраняет всю информацию о модуле доступной для редактирования в администрации, одновременно скрывая ее от витрины магазина. Удаление модуля удалит информацию о модуле магазина. Выбор "Включено" сделает этот модуль общедоступным на страницах, выбранных для магазина.

Порядок сортировки

Порядок сортировки влияет на порядок модуля по отношению к другим модулям в той же позиции. Порядок сортировки 1 для модуля поместил бы его над другим модулем, помещенным в "Content Bottom" с порядком сортировки 2. Если два модуля находятся в одинаковом положении и имеют одинаковый номер порядка сортировки, они будут расположены в алфавитном порядке в соответствии с именем модуля. Модуль с именем "Featured" будет размещен над модулем Specials
