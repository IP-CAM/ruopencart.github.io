Сторонние модули

Расширение магазина

Более 15000 расширений доступны для покупки и загрузки в Магазин расширений OpenCart. Эти модули построены на основе ядра OpenCart для обеспечения дополнительной функциональности и настройки Вашего магазина. Полный список категорий в магазине расширений приведен ниже:

  • Торговые площадки: полезные расширения для магазинов OpenCart, которые действуют как онлайн торговые площадки.
  • Темы: пользовательские темы, чтобы изменить внешний вид Вашего магазина.
  • Языки: дополнительные языковые пакеты доступны для установки. См Добавление нескольких языков научиться добавлять языковые пакеты в свой магазин.
  • Платежные шлюзы: расширения для облегчения дополнительных способов оплаты для обработки транзакций клиентов.
  • Методы доставки: модификации, которые настраивают и устанавливают тарифы на доставку, на инструменты, интегрирующие другие аспекты доставки, не включенные в ядро.
  • Модули: расширения, которые охватывают широкий спектр функций, которые настраивают фронт магазина и администрирование.
  • Итоги заказа: скидки и другие сборы, которые могут применяться к сумме заказа при оформлении заказа в магазине.
  • Ленты продуктов: модули, занимающиеся импортом и экспортом каталога товаров вашего магазина. См. Каналы продуктов для получения дополнительной информации о модулях.
  • Отчеты: дополнительные отчеты для мониторинга активности вашего магазина OpenCart. См. Отчеты для получения дополнительной информации о типах отчетов, уже доступных для основного.

Мы настоятельно рекомендуем Вам найти время, чтобы просмотреть магазин расширений и изучить эти категории, чтобы узнать, что эти сторонние модули могут предложить вашему магазину. Чтобы узнать, какие основные модули уже доступны при установке OpenCart, см. Обзор основных модулей.
