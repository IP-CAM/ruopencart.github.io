Установка / Удаление

Установка и удаление модуля

Версия OpenCart по умолчанию предоставляет несколько основных модулей для администрирования. В разделе «Модуль» некоторые из этих модулей будут установлены и удалены. Мы рекомендуем Вам потратить некоторое время на эксперименты с различными модулями, чтобы определить, какие из них лучше всего подходят для Вашего магазина. Первым шагом в эксперименте является установка модулей. Некоторые модули могут быть удалены по умолчанию, например, модуль Google Talk. Установка модуля очень проста. Вы можете найти модуль и в той же строке нажать "Установка" в Модуле.

Установка

Если модуль установлен, будет возможность "Редактирования" или "Удаление" где "Установка" использовалась.

Удаление

Удаление модуля так же просто, как и установка. Нажатие кнопки "Удалить" под колонкой Действие есть все что нужно для удаления информации о модуле.

Удаление модуля приведет к удалению информации о модуле, выбранной вручную в разделе "Редактировать". Эта информация о модуле не может быть восстановлена после ее удаления путем переустановки.
