Обзор основных модулей

Обзор основных модулей

В последней версии OpenCart есть 26 бесплатных модулей ядра:

Учетная запись Поле Ссылки на учетную запись пользователя (Моя учетная запись, Изменить учетную запись, История заказов и т. д.) Для доступа клиента.
Баннер Добавляет изображение баннера с иконками, связанными с различными продуктами или производителями в магазине.
Бестселлеры Добавляет окно, отображающее бестселлеров, где указана позиция/страница.
Карусель Отображает баннер, по которому можно перемещаться в виде слайд-шоу.
Категория Отображает модуль, содержащий все категории и подкатегории магазина.
Divido Product Page Калькулятор
Рекомендуемые Отображает блок функций, содержащий выбранные продукты.
Фильтр Отображает столбец фильтра для уточнения продуктов.
Google Hangouts Окно чата, используемое с учетной записью Google для взаимодействия с клиентами.
HTML-контент Создает кастом "Welcome" сообщение для клиента для просмотра на главной странице.
Информация Добавляет информационную вкладку, содержащую информационные страницы (О нас, Информация о доставке, Политика конфиденциальности, Условия и положения)
Klarna Checkout
Последние Добавляет последние или недавно добавленные продукты в виде значков продуктов в поле продукта на странице.
Lay-Buy Layout
Log In with PayPal
Login with Amazon
Pay with Amazon
PayPal (Powered by Braintree) Button
PayPal Express Checkout Button
Pilibaba Checkout Button
Sagepay Direct Card Management
Sagepay Server Card Management
Слайд-шоу Отображает баннер, по которому можно перемещаться в виде слайд-шоу.
Специальные предложения Отображает все продукты со скидками в коробке продукта. Специальные предложения могут быть добавлены к продуктам в вкладка Специальные предложения.
Магазин Создает поле для выбора покупателем другого магазина, если есть несколько магазинов, управляемых с помощью OpenCart.
eBay Listing

Все эти модули можно манипулировать с помощью инструментов Макеты, Позиция, Статус и Порядок сортировки в модулях.
