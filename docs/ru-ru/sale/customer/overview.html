Клиенты

Управление клиентами

Владельцы магазинов должны знать, кто их клиенты и как управлять их информацией. В администрации, информация о клиенте должна храниться эффективно, чтобы помнить любые транзакции, сделанные с их учетной записью. Чтобы получить доступ к информации о клиенте, Вы можете войти в панель администрирования магазина OpenCart. Разделы управления клиентами расположены под Продажи > Клиенты.

Для управления информацией о клиентах используются три раздела: клиенты, группы клиентов и черный список IP-адресов.

Клиенты

Когда клиент создает учетную запись или завершает процесс оформления заказа, OpenCart будет автоматически хранить свою информацию в администрации, доступ к которой осуществляется в разделе Продажи>Клиенты>Клиенты. Полный список клиентов будет отображаться на этой странице вместе со следующей информацией:

  • Имя клиентв
  • Эл. почта: Сохраняет адрес электронной почты для подтверждения писем, уведомлений, рассылок и т. д.
  • CГруппа клиента: Специальная группа клиентов, применяющая скидки / специальные предложения и отправляющие групповые электронные письма.
  • Статус: Отказ или подтверждение входа в учетную запись клиента.
  • Одобрение: Утверждает аккаунт клиента после создания.
  • IP-адрес: Адрес, который показывает местонахождение клиента.
  • Дата добавления: Дата создания учетной записи клиента.

Поиск клиента

В случае обнаружения конкретного клиента, введите необходимую информацию о клиенте в полях ниже. Нажатие кнопки "Фильтр" найдет конкретных клиентов, соответствующих введенной информации. Это сэкономит Вам время на просмотр страниц клиентов, чтобы найти конкретного человека. См. Фильтр для получения дополнительной информации о инструмент фильтр.

Добавление нового клиента вручную

Учетные записи клиентов обычно не нужно создавать в панели администрирования, поскольку информация об учетной записи автоматически сохраняется при создании пользователя в магазине. Возможно, Вам придется вручную вводить информацию о клиентах в Клиентах. Нажав на кнопку "Вставить" на странице клиента приведет Вас к форме, где можно заполнить информацию о клиенте.

Вкладка Общая

Вкладка «Общие» запрашивает основную информацию клиента: "Имя", "Фамилия", "Эл. почта", и "Телефон". Пароль должен быть предоставлен для клиента, чтобы получить доступ к сайту в частном порядке. Под "Подтвердить", введите пароль еще раз. Включение "Новостная рассылка" поместит этого клиента в список рассылки по электронной почте. Позже Вы можете классифицировать этого клиента в определенную группу клиентов. Клиент будет разделен на "По умолчанию" если нет определенной группы клиентов. Статус определяет, будет ли этот клиент иметь доступ к логину магазина.

Добавление адреса

Получение адреса покупателя важно для доставки товара покупателю. Щелчок Зеленую кнопку '+' добавит "Адрес 1" адресная форма. Вы можете добавить столько адресов, сколько необходимо, с помощью зеленой кнопки, или удалить их с помощью красной кнопки '-'. "Имя", "Фамилия", "Адрес 1", "Город", "Страна", и "Регион/Область" обязательные поля в форме адреса.

Когда вышеуказанные поля заполнены, нажмите "Сохранить" сохранить форму для администрации. Вам не нужно беспокоиться об IP-адресе клиента для отслеживания транзакций, если только Вам не нужно знать региональное местоположение Вашего клиента.

Раздел «Заказчик» будет в основном использоваться для изменения существующей информации о клиенте, например, для изменения группы клиентов, а не для добавления нового клиента. При редактировании информации о клиенте вы можете изменять общую информацию, включать новые адреса, управлять транзакциями и добавлять бонусные баллы. Под действием, нажав "Редактировать" для клиента, указанного на странице клиента, направит Вас на страницу информации о клиенте. Вы можете изменить информацию о клиенте на вкладке Общие.

Управление транзакциями

На вкладке Транзакция Вы можете вручную добавить транзакцию к клиенту. Для этого введите в описании транзакции уплаченную сумму и нажмите "Добавить транзакцию". Каждая добавленная транзакция будет указана на вкладке Транзакции.

Добавление бонусных баллов

Если Вы решите включить призовые баллы для магазина, баллы могут быть вручную добавлены к определенным покупателям в административной части Вашего магазина. Наградные баллы могут быть добавлены к продуктам, которые будут назначены клиентам, когда они приобретут этот продукт. Если Вы решите добавить бонусные баллы непосредственно клиентам, Вы можете указать причину в описании и количество баллов на вкладке Бонусные баллы. Если Вам нужно вычесть баллы из клиента, Вы можете ввести отрицательное число в поле баллов.

Список будет отображаться каждого события сложения и вычитания в профиле клиента. Баланс будет отображать общую сумму вознаграждений, набранных клиентом. OpenCart вычисляет баланс отрицательных и положительных чисел для вычитания или добавления к итоговой сумме.

IP-адреса

IP-адреса покажут местонахождение компьютерной сети клиента. Владельцам магазинов доступно множество сервисов веб-сайтов, которые дадут местоположение для IP-адреса.

IP-адрес не может быть добавлен или изменен для клиента в OpenCart. IP-адрес будет автоматически получен от клиента OpenCart и добавлен к его информации о клиенте.

Утверждение новых клиентов

Учетные записи клиентов должны быть утверждены, прежде чем они смогут получить доступ к своей учетной записи в магазине. На странице Инфо Панели, количество клиентов, нуждающихся в утверждении, указано в разделе «Обзор». Перейти к Продажи > Клиенты > Клиенты, и найти клиентов, которые перечисляют "Нет" под утвержденной колонкой. Проверьте этих клиентов и нажмите кнопку "Одобрить" в правом углу.
