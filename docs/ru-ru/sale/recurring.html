Периодические заказы

Периодические заказы - это платежи, которые Вы, как администратор, устанавливаете для клиентов, которые должны оплачиваться на постоянной основе.

Они настроены с Каталог > Повторяющиеся профили > Добавить повторяющийся профиль.

Чтобы получить доступ к секции Повторяющиеся заказы, Нажмите на Продажи как показано на рисунке ниже:

Этот раздел позволяет вам фильтровать повторяющиеся платежи, которые вы установили для повторяющихся профилей. Для получения дополнительной информации о профилях - нажмите здесь.

Номер заказа: идентификатор заказа, который связан с повторяющимся платежом.

Ссылка на платеж: идентификатор повторяющегося профиля в PayPal Express (по умолчанию).

Клиент: имя Вашего клиента в OpenCart.

Статус: статус платежа в PayPal.
