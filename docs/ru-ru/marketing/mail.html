Почта

Отправка по электронной почте группе клиентов

Почтовая система в OpenCart позволяет отправлять электронные письма определенным группам клиентов. Вы можете использовать эту функцию для отправки информационных бюллетеней, информации о специальных предложениях или для передачи информации о любом типе магазина для выбора групп покупателей. Группы клиентов особенно полезны для отправки по электронной почте определенным группам людей, а не каждому клиенту, хранящемуся в администрации. Для получения дополнительной информации о том, как создать группу клиентов, см. Группы клиентов. Страница почты может быть доступна в Продажи > Почта.

Выберите группу клиентов в поле "Для:", создайте тему для электронного письма и сообщения и нажмите "Отправить". Это письмо будет отправлено всем клиентам, выбравшим эту группу клиентов.
