Купоны

Купоны

Вы можете создавать и назначать купоны для определенных продуктов или категорий продуктов в разделе Продажи > Купоны.

На странице «Купоны» будет отображен список всех купонов, созданных в администрации.

Вкладка "Общие"

Редактирование или вставка купона приведет администратора к форме, содержащей следующие поля:

  • Название купона: Описательное название купона.
  • Код: Код, который клиент должен ввести в корзину, чтобы использовать купон в магазине.
  • Тип: Скидка может быть фиксированной суммой или процентом от общей суммы.
  • Скидка: Количество, снятое с суммы покупки.
  • Общая сумма: Сумма, которую клиент должен получить до использования купона.
  • Вход для клиентов: Выбор 'да' требует, чтобы клиент вошел в свою учетную запись магазина, чтобы использовать купон. Выбор 'нет' позволит клиенту использовать купон в качестве гостя.
  • Бесплатная доставка: Выбор 'да' предоставляет клиенту бесплатную доставку за использование кода скидки.
  • Категория: При выборе категории товара скидка распространяется на все товары этой категории. См. Категории для получения дополнительной информации о том, как создать новую категорию продукта.
  • Продукты: Отдельные продукты можно выбрать, введя название продукта в предоставленном месте. Чтобы удалить товар из скидки, нажмите красную кнопку минус в списке товаров под ним.
  • Дата начала: Первый день, когда скидка может быть использована.
  • Дата завершения: В последний день скидка может быть использована.
  • Использует за купон: Ограничивает, сколько раз купон может быть использован клиентами.
  • Использует для каждого клиента: Ограничивает, сколько раз конкретный клиент может использовать купон.
  • Статус: Включает или отключает купон для использования в магазине.

Из информации на вкладке «Общие» выше видно, что этот купон дает покупателю скидку в 10 долларов на продукт Acer Aspire.

История купонов

На вкладке История купонов перечислены все варианты использования этого купона в магазине. Следующая информация о каждом использовании будет отображаться:

  • Номер заказа
  • Клиент
  • Дата добавления
  • Количество

Купоны в магазине

Добавление и редактирование купонов в администрации влияет на использование купонов в корзине покупок и оформлении заказа в магазине.

Корзина покупок

Чтобы разрешить использование купонов на витрине магазина, Вы должны включить расширение купона в расширениях Итоги заказа. Для клиента использовать купон "-10.00 Скидка", сначала они должны добавить продукт в корзину со страницы продукта, а затем ввести код в корзину. Они могут выбрать "Используйте код купона", введите код купона в предоставленное место, затем добавьте "Применить купон".

Применение купона вычтет купонную скидку из первоначальной суммы. Вновь обесцененная сумма будет показана под суммой купона (кода) в разделе итогов заказа.
