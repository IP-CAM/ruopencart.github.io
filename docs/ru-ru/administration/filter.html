Фильтр

Фильтр

Фильтр - это удобный инструмент, который регулярно используется в администрации OpenCart. Вы можете ввести в одно из информационных полей для элемента в списке элементов. Нажатие фильтра "отфильтровывает" все элементы в списке, которые не соответствуют критериям, введенным в пространство, оставляя только элементы, которые соответствуют. С помощью фильтра Вы можете быстро найти любую информацию, хранящуюся в форме списка. Продукты, категории, клиенты, отчеты, заказы и многое другое используют фильтр в качестве основного инструмента поиска для поиска элементов в списке.

Например, если Вы хотите найти продукт в списке продуктов, Вы можете использовать "Фильтр" в качестве ярлыка для поиска этого продукта. Это может быть полезно при наличии большого количества товара, и Вы хотели бы редактировать конкретный товар, не просматривая каждую страницу для него. Окна поиска расположены в первой строке над списком товаров.

На снимке экрана выше, "nikon" вводится в информационное поле "Название продукта". "Фильтр" показывает только то, что соответствует "nikon". Nikon D300 - единственный продукт, представленный на выставке, потому что это единственное название продукта, содержащее слова "nikon" в этом магазине. Если формулировка "nikon" присутствует в нескольких названиях продуктов, кнопка "Фильтр" отобразит все из них.
