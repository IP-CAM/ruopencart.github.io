Перемещение OpenCart на новый сервер

OpenCart можно легко перенести на новый веб-сервер. Шаги перечислены ниже:

  1. Скопируйте все файлы OpenCart с существующего веб-сервера на новый веб-сервер..
  2. Экспортируйте базу данных OpenCart и импортируйте ее на новый сервер.
  3. Отредактируйте config.php и admin/config.php. Эти файлы содержат пути файловой системы к папкам OpenCart, URL-адреса для внешнего интерфейса и администратора OpenCart, а также сведения о доступе к базе данных. Пути и детали базы данных, скорее всего, нужно будет обновить, а URL-адреса нужно будет менять только в случае изменения домена магазина OpenCart.
