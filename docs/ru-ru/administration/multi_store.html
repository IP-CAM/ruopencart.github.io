Создание мульти-магазина

Создание мульти-магазина

OpenCart позволяет управлять несколькими магазинами, используя только одну установку. Если Вы установили OpenCart хотя бы в одно хранилище, Вы можете добавить несколько хранилищ в свою панель администратора без необходимости повторять процесс установки. Для добавления нового магазина Вы должны сначала создать поддомен в Вашей cPanel, а затем добавить магазин в разделе «Настройки» панели администратора.

Создание субдомена в cPanel

Вы можете войти в cPanel их хранилища по умолчанию, чтобы создать поддомен. Вы должны создать новую папку для субдомена в разделе Субдомен. В корневом каталоге свяжите поддомен с путем, по которому было установлено хранилище по умолчанию. Например, если хранилище по умолчанию было установлено в "public_html/opencart", новый поддомен (opencart2) должен быть создан в "public_html/opencart". Мы могли бы добавить больше поддоменов здесь, чтобы создать наш мульти-магазин, следуя этой процедуре.

Когда мы посещаем наш новый поддомен в нашем браузере, отображается магазин по умолчанию. Чтобы настроить новый магазин в этом поддомене, нам нужно посетить административную панель нашего мульти-магазина.

Не беспокойтесь, если магазин по умолчанию отображается там, где должен быть новый магазин, это нормально. Новый магазин не будет виден в расположении субдомена, пока он не будет добавлен на стороне администрирования OpenCart.

Создание нового магазина в Настройках

Панель администратора нового магазина можно посетить на любом из сайтов магазина, по адресу "opencart.mystore.com/admin" или "opencart2.mystore.com/admin". Просто добавьте "/admin" в любой из Ваших магазинов, чтобы получить доступ к панели администратора. Существует только одна панель администратора, которая контролирует все магазины в Вашем мульти-магазине.

Чтобы создать новый магазин в администрировании, зайдите в Система > Настройки и нажмите Вставить. Для добавления магазина требуется заполнение информации вкладок Общие, Магазин, Локальность, Опция, Изображение, и Сервер. В эти разделы вы можете добавить новый шаблон, логотип, валюту, язык и макет. После нажатия кнопки "Сохранить" Вы увидите, что хранилище по умолчанию заменяется новым хранилищем в хранилище этого субдомена.

Кастомизация витрины магазина

Продукты, категории продуктов, клиенты, макеты страниц и многое другое можно редактировать в администрации и настраивать для каждого магазина. Вы можете индивидуально выбирать, какие продукты доступны для каждого магазина, на вкладке Связи при редактировании или создании продукта. Проверка магазинов в этом разделе делает продукт доступным только в этих магазинах. При добавлении или изменении категории продукта вы можете выбрать, в каких магазинах будет отображаться категория, на вкладке Данные. Информация о клиенте и заказе будет автоматически отсортирована в соответствующем магазине на стороне администратора при создании учетной записи или покупке продукта в этом магазине.
