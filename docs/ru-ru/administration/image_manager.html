Менеджер изображений

Менеджер изображений

Менеджер изображений - это инструмент, используемый администрацией для загрузки файлов изображений. Баннеры, изображения продуктов, логотип магазина и производители должны загружать изображения с помощью Диспетчера изображений.

Эту функцию можно открыть на страницах категорий, продуктов, баннеров, настроек магазина и т.д., которые необходимы для редактирования изображения магазина.


Клик откроет новое окно под названием "Менеджер изображений". Клик кнопка «Загрузить» на панели инструментов, которая откроет каталог файлов Вашего компьютера, чтобы напрямую добавить их. Изображения ниже предоставлены для продуктов по умолчанию. Выберите изображение, нажав на него, и нажмите Кнопка «Удалить» на панели инструментов, чтобы удалить ее.
